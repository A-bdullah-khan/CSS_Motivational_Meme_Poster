DESCRIPTION:
A simple motivational meme webpage created using HTML and CSS. This project displays an image with a bold heading and a short humorous motivational caption styled using custom fonts.

FEATURES:
1) Centered meme layout
2) Motivational heading text
3) Short descriptive caption
4) Image display
5) Custom Google Fonts integration
6) External CSS styling

CONCEPTS USED:
1) HTML Document Structure
2) Div Container
3) Headings 
4) Paragraph Tag
5) Image Tag
6) External CSS Linking
7) Google Fonts Integration
8) Basic CSS Styling
